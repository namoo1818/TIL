# Listener
- 프로그래밍에서 Listener란 특정 이벤트가 발생하기를 기다리다가 실행되는 객체
- 이벤트란 특정한 사건 발생
  ex) 버튼 클릭, 키보드 입력, 컨테이너 빌드 완료, 웹어플리케이션 시작, HTTP 요청 수신 등등
- 이벤트 소스란 이벤트가 발생한 근원지(객체)
