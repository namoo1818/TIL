- servlet-context : 웹과 관련된 설정이 들어있다
- root-context : 웹 이외의 설정이 들어있다  

AInterceptor, BInterceptor, CInterceptor를 차례대로 servlet-context.xml에 등록할 경우

```
A : preHandle
B : preHandle
C : preHandle
C : postHandle
B : postHandle
A : postHandle
C : afterCompletion
B : afterCompletion
A : afterCompletion
```
