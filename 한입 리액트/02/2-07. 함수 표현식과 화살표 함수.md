## 호이스팅
> **함수 선언식**으로 만들어진 함수는 프로그램 실행 전에 코드의 최상단에 끌어올라가진다</br>
- 주의 : 함수 표현식은 호이스팅 안된다
```js
console.log(helloB());

let helloA = function () {
  return "안녕하세요 여러분";
}; //함수 표현식

function helloB() {
  return "안녕하세요 여러분";
} //함수 선언식
```
```
안녕하세요 여러분
```

## 화살표 함수
- 주의 : 화살표 함수는 호이스팅이 안된다
```js
let helloA = () => "안녕하세요 여러분"; //화살표 함수

console.log(helloA());
```
```
안녕하세요 여러분
```
