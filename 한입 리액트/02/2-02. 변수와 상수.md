# 변수
- 프로그램이 실행되는 도중에 계속해서 바뀌는 값을 저장하기 위한 것
## 변수 선언
```
let age = 20;
console.log(age);
age = 30;
console.log(age);
```
```
20
30
```

## 변수명 규칙
1. "_", "$" 이외 기호 사용 불가
2. 숫자가 맨 앞에 올 수 없다
3. 예약어 사용 불가

## var와 let
### var
> 변수 중복 선언 가능
### let
> 변수 중복 선언 불가능

# 상수
- 변하지 않는 값
## const
```
const age; // error: 상수는 선언과 동시에 초기화되어야한다
const age = 20;
age = 30; // error: 선언 이후에 값 변경 불가
```
